ColorFlex Trade Show (offline shell)

This is the wallpaper-only offline demo scaffold served over http://localhost (not file://). Wi‑Fi can stay off once the app and data are on disk.

Phase 1 only: static placeholder. The ColorFlex webpack bundle, demo bootstrap globals, and frozen wallpaper snapshot are not wired yet — that comes in later phases. No Shopify, Liquid, or CFM integration here.

When you run npm run serve:trade-demo, you should see this page at the server root.

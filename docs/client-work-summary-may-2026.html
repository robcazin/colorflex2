Saffron Cottage / ColorFlex on Shopify

Work Summary  ·  May 1 – May 27, 2026

This report summarizes development work on the Saffron Cottage Shopify store and the ColorFlex customization platform that powers it. A separate Mohawk Room demo (for trade shows) shares the same engine but is not the main shop.

The big picture

Most of May focused on the live Shopify store — where customers customize wallpaper and fabrics, save designs, and order custom samples. In parallel, we built a standalone Mohawk Room preview demo for carpet + room coordination, hosted outside the main checkout flow.

Part 1 — Fixing the live Shopify store (early May)

These were production fixes on saffroncottage.shop — issues customers would notice if they broke.

Custom sample & Add to cart

When someone designs a pattern in ColorFlex and opens the custom sample product page, they must add their design to the cart with the correct colors and thumbnail.

  • Add to cart sometimes did nothing — caused by invalid nested forms on the product page. The buy button was effectively outside the real cart form.
  • Cart thumbnail and pattern details — improved how the saved pattern image and color data travel into the cart.
  • Stronger protection of custom line-item data — so theme scripts are less likely to strip ColorFlex’s saved color information.

In plain terms: customers can again add a customized sample to the cart and trust that what they see is what gets ordered.

Deploy confidence on Shopify

  • Added a simple version marker (CF-REV) in the theme so you can confirm new code loaded after an update.
  • Improved deployment workflow so fixes reach the live theme faster and more consistently.

Mohawk work in relation to Shopify

Mohawk Room extends the same ColorFlex codebase: carpet thumbnails drive pattern colors and a furnished room preview. It can be enabled on Shopify ColorFlex pages when desired. The standalone demo URL is packaged for trade shows — not a replacement for the Shopify store.

Part 3 — Standalone Mohawk demo (late May)

A static demo (hosted separately from Shopify) lets Mohawk show carpet + wallpaper + room preview without a full store session. It shares the ColorFlex engine but has no cart or checkout — a showroom version, not the main shop.

What to tell stakeholders about Saffron Cottage

  • We fixed critical Shopify checkout issues on the custom sample product — add to cart and cart line details work reliably again.
  • We improved how deployed code is verified on the live theme, so updates are traceable.
  • ColorFlex’s core app on Shopify continued to evolve — better previews, data loading, and foundation for carpet-driven color coordination.
  • Mohawk Room demo work shares the same engine; the live store remained the priority for production stability in early May.

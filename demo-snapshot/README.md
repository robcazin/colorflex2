# Trade-show demo snapshot (`demo-snapshot/`)

Generated by `npm run build:trade-show-snapshot`.

Contains **`data/collections.json`** (wallpaper collections from `src/assets/collections.json`), **`data/mockups.json`**, and **`data/colors.json`**.

Raster assets (thumbnails, layers, mockup images) are **not** copied here; the trade-show shell sets `COLORFLEX_DATA_BASE_URL` to canonical cf-data so the app loads them from the same URLs as production.

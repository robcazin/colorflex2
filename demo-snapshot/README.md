# Trade-show demo snapshot (`demo-snapshot/`)

Generated by `npm run build:trade-show-snapshot`.

Contains **`data/collections.json`** (wallpaper collections from `src/assets/collections.json`), **`data/mockups.json`**, **`data/colors.json`**, and **`data/furniture-config.json`** when the script emits it.

Raster assets (thumbnails, layers, mockup images) are **normally not** copied here; the trade-show app sets `COLORFLEX_DATA_BASE_URL` to **`/cf-data`** so the local server can serve the same on-disk layout as production (see `trade-show-local/server.js`). When the local `cf-data/` folder is missing, the server can proxy pattern assets from Backblaze under `/cf-data` so canvas pipelines still get CORS-safe responses.

Optional: `SNAPSHOT_DOWNLOAD_ASSETS=1 npm run build:trade-show-snapshot` downloads rasters into this tree (large).
